# Context: Active Development

Mode: Active development
Focus: Implementation, coding, building features

## Behavior
- Code first, explain after
- Prefer working solutions over perfect ones
- Run tests after changes
- Make atomic commits

## Priorities
1. Make it work
2. Make it right
3. Make it clean

## Preferred Tools
- Edit, Write for code changes
- Shell/Bash for running tests and builds
- Grep, Glob for finding code
- A browser-automation tool (e.g. Playwright MCP) for visual verification
